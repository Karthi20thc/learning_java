
# âœ… **1. Control Flow Keywords**

These keywords control the execution flow of a program.

### **ðŸ”¹ if / else**

Used to make decisions.

```java
if (age > 18) {
    System.out.println("Adult");
} else {
    System.out.println("Minor");
}
```

---

### **ðŸ”¹ switch, case, default**

Used when checking multiple values.

```java
switch (day) {
    case 1: System.out.println("Mon"); break;
    case 2: System.out.println("Tue"); break;
    default: System.out.println("Unknown");
}
```

---

### **ðŸ”¹ do / while**

Loop that runs at least once.

```java
int i = 1;
do {
    System.out.println(i);
    i++;
} while (i <= 5);
```

---

### **ðŸ”¹ for**

Used for loops.

```java
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}
```

---

### **ðŸ”¹ break**

Stops a loop or switch.

```java
for(int i=0;i<10;i++){
    if(i==5) break;
}
```

---

### **ðŸ”¹ continue**

Skips current loop iteration.

```java
for (int i = 1; i <= 5; i++) {
    if (i == 3) continue;
    System.out.println(i);
}
```

---

### **ðŸ”¹ return**

Exits a method and optionally returns value.

```java
return x + y;
```

---

    case 1 -> 10;
